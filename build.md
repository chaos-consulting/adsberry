# Bauanleitung - How to Build an ADS-Berry Pi
## Teileliste
* Raspberry Pi B, B+, 2 oder 3
* (Micro) SD Speicherkarte 4GB oder mehr
* Buck Boost Converter
* Passive PoE Splitter
* 30cm USB Verlängerung
* Micro USB Kabel
* DVB-T Stick 820T
* Gehäuse
* MCX -> SMA Adapter
* 1090MHz Bandpassfilter
* Kabeldurchführung
* RJ45 Durchführung
* 1m SMA -> N Kabel
* N Buchse
* 40cm Kupferdraht
* 4 3mm Schrauben + Muttern
* Schrumpfschlauch
* Montagekleber
## Werkzeugliste
* Stufenbohrer
* Lötkolben
* Zange
* Seitenschneider
* Abisolierzange (optional)
* PH2 Schraubendreher
